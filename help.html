关于 Aero/Mica/亚克力

Aero 仅在 Windows 7 可用，Mica 仅在 Windows 11 (Build 22523 及以上) 可用，亚克力效果仅在 Windows 10 (1803及以上) 可用。

关闭此效果

两种方法可关闭此效果：
1. 将纯色图片设为壁纸
2. 或以兼容模式运行，将兼容模式设为除 Windows 7, Windows 10 和 Windows 11 以外的操作系统


问题排查

**在开始之前，请先确保计算机上安装有 .net 6运行时 **

api-ms-win-crt-runtime-l1-1-0-dll缺失

安装Microsoft Visual C++ Redistributable

检查更新失败

确保网络连接正常，访问此链接检查更新服务的可用性
检查计算机 %APPDATA%/中育飞车助手/Installer.exe 是否成功下载

标题等中文乱码

由于字符编码更改，自 3.1.1 以下的用户在更新后会遇到中文字符乱码问题，请前往手动更改 Program Files/中育飞车助手/config.ini 字符编码为 UTF-8
